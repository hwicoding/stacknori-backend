## 서버 커밋 메시지 규칙

NAS 서버에서 작업할 때 커밋 메시지는 아래 형식을 사용합니다.

```
YYMMDD > server > 작업내용 한 줄 요약
```

- `YYMMDD`: 커밋 시점 기준 6자리 날짜 (예: 251127)
- `server`: 서버 환경에서 만든 커밋임을 나타내는 고정 키워드
- `작업내용 한 줄 요약`: 이번 변경을 1줄로 간단하게 설명

### 예시
```
251127 > server > Nginx SSL 인증서 갱신 스크립트 추가
251201 > server > Runner systemd 서비스 로그 경로 수정
```

커밋/푸시 요청 시 항상 위 규칙을 참고하세요.
